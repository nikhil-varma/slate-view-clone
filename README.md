# SlateClone

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 7.3.8.

## Pre-requisites
Open the project in Visual Code or any editor of choice

If the editor comes with a pre-loaded terminal follow the dependency installation steps. Else 
open the terminal from the project folder (in the folder containing `package.json`) and then follow the dependency installation steps

### Dependency Installation 
Before running the project execute the following command to include the dependencies
`npm i`

Incase the process throws an error try `npm cache clean -f ` and execute the previous command again.

## Development server

Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`. The app will automatically reload if you change any of the source files.


## Basic Troubleshooting
    `ng` is not recognized as the name of a cmdlet or command

If encountered, please run the following command `npm i -g @angular/cli`

    Dependency installation failure or EPERM error
Please make sure that the command prompt or the editor is provided adming privileges to run
